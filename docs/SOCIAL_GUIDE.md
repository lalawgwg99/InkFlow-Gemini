# 📱 SocialContent-AI：社群經營者的終極指南

嗨！我是這個工具的重度使用者。如果你像我一樣，每天需要在 **IG (發乾貨卡片)**、**X (發碎碎念貼文串)** 和 **FB (發長文)** 之間切換，這個工具會是你的救星。

它不再只是把文章轉成微信公眾號格式，而是能把你的 `Markdown` 原稿，一鍵變成適合各平台的「原生爆款格式」。

---

## 🚀 第一步：準備你的軍火庫 (設定一次就好)

首先，我們要給引擎加燃料。現在核心是 **Google Gemini**，它聰明又便宜（甚至免費）。

1. 去 Google AI Studio 申請一個 API Key。
2. 告訴工具你的 Key：

    ```bash
    # 在 Windows PowerShell 設定環境變數
    $env:GEMINI_API_KEY="你的_API_KEY"
    ```

---

## 📸 場景一：製作 IG/FB 的「高質感乾貨卡片」

這是我最常用的功能。寫好一篇 Markdown 筆記，直接轉成漂亮的圖片卡片。

**指令：**

```bash
# --mode card: 啟動卡片模式
# --theme: 選擇風格 (magazine-dark, minimalist-gray, tech-neon)
md2wechat convert my_post.md --mode card --theme magazine-dark -o preview.html
```

**發生了什麼事？**

1. 工具會自動把你的文章切成 10 張左右的投影片 (Slide)。
2. 生成一個 `preview.html` 檔案。
3. **怎麼用？**
    - 雙擊打開 `preview.html`。
    - 你會看到精美的卡片排列。
    - **截圖大法**：直接用截圖軟體 (Win+Shift+S) 把每一張卡片截下來，丟到 IG 發布。
    - **進階**：使用瀏覽器的「列印 -> 另存 PDF」或截圖插件。

**風格推薦：**

- 💄 **黑金雜誌 (`magazine-dark`)**：適合講商業思維、人生哲學，高級感拉滿。
- 🌿 **極簡灰調 (`minimalist-gray`)**：適合文青、心情語錄、攝影作品。
- ⚡ **賽博霓虹 (`tech-neon`)**：適合講 AI 工具、加密貨幣、黑科技。

## 🧵 場景二：發布 X (Twitter) 的「長貼文串 (Thread)」

X 最討厭長篇大論，喜歡「貼文串」。以前手動切分超級累，現在一鍵搞定。

**指令：**

```bash
# --mode thread: 啟動貼文串模式
md2wechat convert my_thought.md --mode thread
```

**輸出範例：**

```text
=== Twitter Thread Output (5 tweets) ===

🧵 1/5
自律是個謊言。
大多數人以為自律是「強迫自己做不喜歡的事」。
錯了。
真正的高手，是透過「設計環境」讓行動變得自動化。
(這裡有100字...)

---

2/5
想像一下，如果你把手機放在另一個房間...
(這裡有200字...)

---

3/5
...
```

**怎麼用？**

- 直接把終端機輸出的內容，複製貼上到 X 的發文框（或是使用 Typefully 之類的工具）。
- 它已經幫你斷好句、編好號 (1/5, 2/5)，甚至幫你保留了 Emoji。

## 💡 專業玩家的小撇步

1. **預覽 Hover 特效**：在 `card` 模式生成的 HTML 中，試著把滑鼠移上去，你會看到背景在流動，標題在發光。這雖然不能直接截圖，但在 demo 給客戶看時非常帥。
2. **Markdown 格式**：
    - 使用 `---` (分隔線) 來強制卡片換頁。
    - 使用 `**粗體**` 來標示重點，在黑金主題中會變成金色字體。
    - 使用 `> 引用` 來做金句，樣式會特別突出。
